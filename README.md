# 벽돌깨기 (Brick Breaker)

브라우저에서 실행하는 클래식 벽돌깨기 게임입니다.  
**같은 WiFi에서 친구와 공유하거나, 인터넷에 배포해서 누구나 접속해 플레이할 수 있습니다.**

---

## 다른 사람이 접속해서 플레이하는 방법

### 방법 1: 같은 WiFi에서 공유 (로컬 네트워크)

1. 프로젝트 폴더에서 서버를 실행합니다.

   ```bash
   npm start
   ```

   (Node.js가 없다면 [nodejs.org](https://nodejs.org)에서 설치)

2. 터미널에 출력된 **다른 사람 접속** 주소(예: `http://192.168.0.5:3000`)를 복사해서 친구에게 보냅니다.

3. 친구는 **같은 WiFi**에 연결된 상태에서 그 주소를 브라우저에 입력하면 게임을 할 수 있습니다.

---

### 방법 2: 인터넷에 배포 (누구나 접속 가능)

**Vercel로 배포 (무료, 추천)**

1. [Vercel](https://vercel.com)에 가입하고, 터미널에서 Vercel CLI를 설치합니다.

   ```bash
   npm i -g vercel
   ```

2. 프로젝트 폴더에서 배포합니다.

   ```bash
   vercel
   ```

   처음에는 이메일 로그인, 프로젝트 이름 등을 묻습니다. 그대로 진행하면 됩니다.

3. 배포가 끝나면 `https://프로젝트이름.vercel.app` 같은 **공개 URL**이 나옵니다.  
   이 주소를 친구에게 보내면, **어디서든** 그 링크만 열면 게임을 할 수 있습니다.

**GitHub Pages로 배포**

1. 이 프로젝트를 GitHub 저장소에 올립니다.
2. 저장소 **Settings → Pages**에서 Source를 **main** 브랜치, 폴더를 **/ (root)** 로 설정 후 저장합니다.
3. 몇 분 후 `https://사용자이름.github.io/저장소이름/` 주소로 접속할 수 있습니다.

---

## 로컬에서만 실행할 때

```bash
# Node.js
npm start
# 브라우저에서 http://localhost:3000

# 또는 Python 3
python3 -m http.server 8080
# 브라우저에서 http://localhost:8080
```

---

## 조작법

- **마우스**: 화면에서 좌우로 움직이면 패들이 따라 움직입니다.
- **키보드**: ← → 방향키로 패들을 조작합니다.

## 게임 규칙

- 공을 패들로 받아 벽돌에 맞혀 깨세요.
- 벽돌 하나당 10점입니다.
- 공을 떨어뜨리면 생명이 1 감소하며, 생명이 0이 되면 게임 오버입니다.
- 모든 벽돌을 깨면 승리하고, "다음 레벨"로 더 빠른 공에 도전할 수 있습니다.

## 파일 구성

- `index.html` - 게임 화면 구조
- `style.css` - 스타일
- `game.js` - 게임 로직 (패들, 공, 벽돌, 충돌, 점수)
- `server.js` - 같은 네트워크 접속용 서버 (선택)
- `vercel.json` - Vercel 배포 설정 (선택)
